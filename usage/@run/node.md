## @run/node.ts

Run your node application

### Usage

**Required Fields**

-   mainFile

    > The mainfile. It get's executed on run

**Optional Fields**

-   arguments

    > The arguments to the program, you're running. Type: Array

-   envFile

    > The environment file. It get's parsed and passed into the environment.
    > 
    > Fileformat: .env
